#  DSA Visualizer

A web-based Data Structures & Algorithms Visualizer that helps students understand sorting algorithms through interactive animations.

---

##  Problem Statement
Students often struggle to understand how sorting algorithms work step-by-step internally.  
This project solves this by visualizing algorithms in real-time.

---

## Features
- Visual representation of sorting algorithms
- Speed control
- Algorithm selection
- Real-time animation

---

##  Tech Stack
- React.js
- JavaScript
- HTML
- CSS

---

##  Algorithms Implemented
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

---
